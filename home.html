VoxelCube

A cozy group, with cozy people, and ... I made a perfect icon. The group originally started as TechnoGamers (cringe name ikik), then was renamed to Ferreto Studios. Recently, we changed the name to VoxelCube as there was too much of me (Ferretosan is writing this btw) and not enough of other people.

This website is also open source, so you should check it out on GitHub (in the footer)! Have fun with the code, and try to find some of the leftover developer features we never removed!
